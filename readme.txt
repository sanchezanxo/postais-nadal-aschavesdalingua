Plugin para crear e compartir postais de Nadal personalizadas en galego.

== Descrición ==

Postais de Nadal permite aos usuarios crear postais navideñas personalizadas cunha interface sinxela e moderna.

**Características:**

* Editor visual con canvas interactivo
* Soporte para texto personalizado con múltiples fontes
* Stickers decorativos
* Tres formatos: Postal (3:2), Instagram (4:5), Historia (9:16)
* Descarga directa e envío por email
* Compartir en WhatsApp (móbil)
* Panel de administración con estatísticas
* Exportación CSV dos rexistros
* Rate limiting para evitar abusos
* Totalmente en galego

== Instalación ==

1. Sube a carpeta `postais-nadal-aschavesdalingua` ao directorio `/wp-content/plugins/`
2. Activa o plugin desde o menú 'Plugins' en WordPress
3. Configura as opcións en 'Postais Nadal' > 'Configuración'
4. Sube as imaxes base nas carpetas correspondentes ou usa a Biblioteca de Medios
5. Usa o shortcode `[postais_nadal]` para mostrar o botón no frontend

== Uso ==

**Shortcode básico:**
`[postais_nadal]`

**Con texto personalizado:**
`[postais_nadal texto="Crea a túa postal!"]`
