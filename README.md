# Postais de Nadal

Plugin de WordPress para crear e compartir postais de Nadal personalizadas en galego.

## Características

- Editor visual con canvas interactivo
- Texto personalizado con múltiples fontes
- Stickers decorativos
- 3 formatos: Postal (3:2), Instagram (4:5), Historia (9:16)
- Descarga, email e WhatsApp
- Panel de administración con estatísticas
- Exportación CSV

## Instalación

1. Sube a carpeta ao directorio `/wp-content/plugins/`
2. Activa o plugin en WordPress
3. Configura en **Postais Nadal > Configuración**
4. Usa o shortcode `[postais_nadal]`

## Uso

```
[postais_nadal]
[postais_nadal texto="Crea a túa postal!"]
```

## Requisitos

- WordPress 5.0+
- PHP 7.4+
